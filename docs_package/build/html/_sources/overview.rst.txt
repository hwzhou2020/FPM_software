
Overview
========

This software provides a graphical user interface and a set of algorithmic tools
for performing Fourier Ptychographic Microscopy (FPM) image reconstruction.

Features include:
- Loading and visualizing raw and processed microscopy data.
- Running reconstruction algorithms.
- Adjusting system and algorithm parameters via a GUI.
- Saving and exporting reconstructed images.

Project Structure:
------------------
- **main.py**: Application entry point.
- **Algorithms/**: Core FPM reconstruction algorithms.
- **Utilities/**: Utility functions for loading, display, message handling, etc.
- **WindowUI/**: Qt Designer-generated and manually customized UI classes.

This documentation includes:
- Usage instructions
- Module-level API reference
- Class and function descriptions
